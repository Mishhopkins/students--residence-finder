# Msavi Campus Residence Finder

###### This Web application helps landlords and owners to register individaul home or apartment to assist students (comrades) in finding the perfect rental home or property types. Also they can find their next rental from search view in a targeted area, and within a given budget.

## Built with Core PHP, PHP PDO, Bootstrap4, Jquery, MYSQL.
